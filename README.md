# Mikuse Team

## Jak pobrać repo:
`git clone git@github.com:my-rspec/mocking-hell-mikuse-team.git`

## Praca na branchach
Utworzenie nowego brancha z githuba: 

`git checkout -b <imie> origin/<imie>`

Następnie przy wysłaniu kodu na serwer:

`git push -u origin <imie>`
